# Git Links #

## Best Practices ##

- Pull then Push
- Commit Often
- Push infrequently

## Helpful Commands ##

- git diff
  - see changes made to files
  - press 'q' to exit
- git diff --staged
  - see changes made to staged files
- git reflog
  - see all things you have done in git
